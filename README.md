## Description

[Nest.js](https://github.com/nestjs/nest) Stockx Starter

## Installation

```bash
# clone the repository
git clone $repo_url

# go to project dir
cd ./stockx-microservice-starter

# install
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
